# git-standup 11 "Jun 2015" git-standup "User Manuals"

## SYNOPSIS

`git standup`

## DESCRIPTION

Views all commits made in the last week.

## OPTIONS

`-h`, `--help`
: Prints help.

`-v`, `--version`
: Prints the version.
